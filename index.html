주요 저장소 소개

🧠
ComfyUI
노드 기반 AI 생성 워크플로우 GUI 본체. 이미지·비디오·오디오 생성 파이프라인을 코딩 없이 구성. SD1.5 / SDXL / Flux / Wan 등 다양한 모델 지원.
Core Engine
🔌
ComfyUI-Manager
커스텀 노드 및 모델 통합 관리 도구. UI 내에서 노드 설치·업데이트·제거. 누락 노드 자동 감지 및 설치 제안. 스냅샷/복원 기능.
Node Manager
⬇️
ComfyUI-Downloader
UI에서 URL 입력만으로 모델을 올바른 폴더에 직접 다운로드. 8개 병렬 연결 고속 다운로드. 실시간 진행률 표시 및 취소 지원.
Model Downloader

Slide 03

ComfyUI 설치 — Linux

# 1. 저장소 클론
git clone https://github.com/Comfy-Org/ComfyUI.git
cd ComfyUI

# 2. 가상환경 생성 및 활성화
python3 -m venv venv
source venv/bin/activate

# 3. PyTorch 설치 (CUDA 13.0 기준)
pip install torch torchvision torchaudio \
  --extra-index-url \
  https://download.pytorch.org/whl/cu130

# 4. 의존 패키지 설치
pip install -r requirements.txt

# 5. 실행
python main.py --listen 127.0.0.1
주요 실행 옵션
--listen 127.0.0.1로컬 전용 접속
--listen 0.0.0.0외부 접속 허용
--port 8188포트 지정 (기본값)
--preview-method auto생성 중 미리보기
--cpuCPU 모드
SSL 인증서 오류 방지
# run_gpu.sh 에 추가
export SSL_CERT_FILE=/etc/ssl/certs/ca-certificates.crt
export REQUESTS_CA_BUNDLE=/etc/ssl/certs/ca-certificates.crt
접속 주소
http://127.0.0.1:8188

마우스 조작 & 워크플로우 팁

마우스 조작
빈 공간 LMB Drag 캔버스 패닝
Scroll Wheel 줌 인/아웃
LMB Drag 빈 공간 박스 선택 (영역 드래그)
출력 포트 Drag 연결선 생성 → 입력 포트에 드롭
포트 → 빈 공간 Drop 호환 노드 검색창 팝업
연결선 더블클릭 Reroute 노드 삽입
위젯 RMB Convert to Input / Widget
이미지 노드 RMB 이미지 저장 / 열기
워크플로우 팁
PNG에 워크플로우 내장
생성된 PNG/WebP에 워크플로우 전체가 메타데이터로 포함. 이미지를 캔버스에 드래그&드롭하면 자동 복원.
변경된 노드만 재실행 (스마트 캐싱)
미변경 노드는 재실행 없이 캐시 사용. Sampler만 바꾸면 Checkpoint 로딩 생략 → 반복 작업 고속.
Primitive 노드로 값 공유
동일 Seed/CFG를 여러 KSampler에서 공유할 때 Primitive 노드 하나를 여러 입력에 연결.
Mute vs Bypass 구분
Ctrl+M Mute: 완전 비활성화. Ctrl+B Bypass: 입력→출력 패스스루. A/B 비교엔 Bypass.
Badge Mode & Link Render Mode
Settings → Badge Mode: Show All → 노드 출처 표시. Link Render Mode: Straight → 선 직선화로 가독성 향상.
모델 핫 리로드
실행 중 models/ 폴더에 파일 추가 후 단축키 R(Refresh)를 누르면 목록에 즉시 반영.
Slide 07

CLI 실행 옵션

VRAM 최적화
# VRAM 크기에 따라 선택
python main.py --lowvram # 6GB 이하
python main.py --normalvram # 8GB (기본값)
python main.py --highvram # 12GB+ · 최대 성능
python main.py --novram # CPU 메모리 활용
python main.py --cpu # GPU 없이 CPU만 사용
성능 & 정밀도
python main.py --fp16 # FP16 (기본값)
python main.py --bf16 # BF16 (Ampere+ 권장)
python main.py --fp32 # FP32 (느리지만 정확)
python main.py --xformers # 15-25% 속도 향상
python main.py --preview-method auto # 프리뷰 방식
네트워크 & 디버그
python main.py --listen 127.0.0.1 # local only
python main.py --listen 0.0.0.0 # LAN 접근 허용
python main.py --port 8188 # 포트 변경
python main.py --verbose # 상세 로그
python main.py --disable-auto-launch # 브라우저 자동 실행 방지
extra_model_paths.yaml — 다중 모델 경로
# ~/.config/comfyui/extra_model_paths.yaml
my_models:
  base_path: /mnt/models/
  checkpoints: checkpoints/
  loras: loras/
  vae: vae/
  controlnet: controlnet/
  text_encoders: text_encoders/
  diffusion_models: diffusion_models/
  clip_vision: clip_vision/
extra_model_paths.yaml 장점: 모델 파일을 ComfyUI 폴더 밖에 두고 경로만 등록. 여러 ComfyUI 인스턴스에서 동일 모델 파일 공유 가능. 시작 시 --extra-model-paths-config 플래그로 지정.
Linux xformers: RTX 계열 GPU에서 유효. pip install xformers 별도 설치 필요 (ComfyUI venv 활성화 후). CUDA 버전 맞아야 설치 성공.

class MyNode:

  # ① 입력 타입 정의 (필수)
  @classmethod
  def INPUT_TYPES(s):
    return {
      "required": {
        "image":    ("IMAGE",),
        "steps":    ("INT", {"default": 20, "min": 1, "max": 100}),
        "strength": ("FLOAT", {"default": 0.8, "min": 0.0, "max": 1.0}),
        "text":     ("STRING", {"multiline": True}),
      },
      "optional": {
        "mask": ("MASK",),  # 없어도 실행 가능
      }
    }

  # ② 출력 타입 (필수)
  RETURN_TYPES = ("IMAGE", "STRING")
  RETURN_NAMES = ("result", "info")  # 선택

  # ③ 실행 메서드 이름 (필수)
  FUNCTION = "process"

  # ④ UI 카테고리 (필수)
  CATEGORY = "MyNodes/Image"

  # ⑤ 실제 실행 메서드
  def process(self, image, steps, strength, text, mask=None):
    result = image * strength
    info = f"steps={steps}"
    return (result, info)  # RETURN_TYPES 순서와 일치
INPUT_TYPES 구분
required — 반드시 연결 필요
optional — 연결 없어도 실행 가능
hidden — UI에 숨김, 내부 전달용
RETURN_TYPES
튜플로 반환 타입 선언
실행 메서드도 동일 순서 튜플 반환
출력 없으면 RETURN_TYPES = ()
OUTPUT_NODE = True

import torch

class BrightnessAdjust:

  @classmethod
  def INPUT_TYPES(s):
    return {
      "required": {
        "image": ("IMAGE",),
        "brightness": ("FLOAT", {
          "default": 1.0,
          "min": 0.0,
          "max": 3.0,
          "step": 0.05,
        }),
      }
    }

  RETURN_TYPES = ("IMAGE",)
  FUNCTION     = "adjust"
  CATEGORY     = "image/adjust"

  def adjust(self, image, brightness):
    result = torch.clamp(image * brightness, 0.0, 1.0)
    return (result,)

  model_type = request.match_info["model_type"]
  filename   = request.match_info["filename"]
  full_path  = folder_paths.get_full_path(
      model_type, filename)
  if not full_path or not os.path.exists(full_path):
    return web.json_response(
        {"error": "파일 없음"}, status=404)
  os.remove(full_path)
  return web.json_response(
      {"status": "deleted"})

@routes.post("/my_manager/download")
async def trigger_download(request):
  data     = await request.json()
  url      = data.get("url")
  dest_dir = data.get("dest")
  filename = data.get("filename")

  if not all([url, dest_dir, filename]):
    return web.json_response(
        {"error": "url/dest/filename 필수"},
        status=400)

  # 비동기 다운로드 태스크 실행 후 즉시 응답
  asyncio.create_task(
      _download_task(url, dest_dir, filename))
  return web.json_response({
      "status": "started",
      "filename": filename
  })

WebSocket 실시간 통신

서버 → 브라우저 푸시 (api.py)
async def _download_task(url, dest_dir, filename):
  dest_paths = folder_paths.folder_names_and_paths
  save_dir   = dest_paths[dest_dir][0][0]
  save_path  = os.path.join(save_dir, filename)

  async with aio.ClientSession() as session:
    async with session.get(url) as resp:
      total      = int(resp.headers.get(
                       "Content-Length", 0))
      downloaded = 0
      with open(save_path, "wb") as f:
        async for chunk in \
            resp.content.iter_chunked(1024*1024):
          f.write(chunk)
          downloaded += len(chunk)
          pct = round(downloaded/total*100, 1)

          # 진행률 실시간 브로드캐스트
          PromptServer.instance.send_sync(
            "my_manager/progress",
            {"filename": filename, "progress": pct}
          )

  # 완료 이벤트 전송
  PromptServer.instance.send_sync(
    "my_manager/done",
    {"filename": filename, "path": save_path}
  )
